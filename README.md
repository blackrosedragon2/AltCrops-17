# AltCrops-17
Steps to Run Frontend:
1. ```cd frontend```
2. ```npm init```
3. ```npm run start```

## Steps to run Backend:
1  ```cd backend```
2. ```npm init```
3. ```npm run start ```

## Steps to start Middleware:
1. ```cd middleware```
2. ```python app.py```
